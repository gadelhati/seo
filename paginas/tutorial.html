CONFIGURAR WILDFLY

  1. Na aba Servers > No servers are available. Click this to create a new server...
    2. ou
  1. Na aba Servers > Botão direito: New > Server
  2. Janela: New Server > Select the server type: Red Hat JBoss Middleware > JBoss AS, Wildlfy, & EAP Server Tools > Next
  3. Janela: New Server > I accept the terms of the license agreement > Finish
  4. Janela: Security Warning > Install anyway
  5. Janela: Software Updates > Restart Now
  6. Janela: JBoss Tools Usage > Yes

CONFIGURAR PROJETO WILDFLY

  1. Na aba Servers > No servers are available. Click this to create a new server...
    2. ou
  1. Na aba Servers > Botão direito: New > Server
  2. Janela: New Server > Select the server type: JBoss Community > Wildlfy 13 > Next > Next
  3. Janela: New Server > Home Directory: Browse... [selecionar a pasta do WILDLFY, wildfly-13.0.0.Final]
  4. Janela: New Server > Botão: Installed JREs...
  5. Janela: Preferences > Botão: Search...
  6. Janela: Directory Selection > [selecionar a pasta do JDK, C:\Program Files\Java\jdk1.8.0_181] > Selecionar pasta
  7. Janela: Preferences > [selecionar o novo JRE criado] > Apply and Close
  8. Janela: New Server > Alternate JRE > [selecionar o novo JRE criado] > Finish
